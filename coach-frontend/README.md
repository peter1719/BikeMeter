# Coach Frontend

## 建置說明

1. 安裝依賴
```
npm install
```

2. 啟動開發伺服器
```
npm run dev
```

3. 建置生產版本
```
npm run build
```

## 環境要求
- Node.js 16.x 或更高版本
- npm 7.x 或更高版本
